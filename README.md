# Töötundide loendur
Kõigepealt natuke taustast. Mõtte projekti jaoks sain sellest rakendusest (endal küll plaanis PC versioon): https://play.google.com/store/apps/details?id=ro.mb.mastery5 See rakendus võimaldab arvet pidada erinevate oskuste arendamiseks kulutatud aja üle. Põhiidee on see, et mingis oskuses meisterlikkuse saavutamiseks tuleb seda harjutada 10 000 tundi (üsna kõrge eesmärk muidugi…). Samas on mõned funktsioonid, millest ma selle rakenduse juures puudust tunnen, näiteks sisse ehitatud taimer ja täpsem statistika.

Minu programmi kirjeldus:
1. Kasutaja saab sisestada erinevaid oskusi, mille kohta ta aega tahab arvestada
2. Sissekannete tegemine kas käsitsi (sel kuupäeval tegin seda tegevust x minutit) või taimeri abil
3. Sissekannete juures konkreetsem tegevuse kirjeldus (nt lugesin sellel teemal raamatut, harjutasin oskust praktikas vms)
4. Võimalus sissekandeid muuta/kustutada
5. Programm väljastab kasutajale erinevat statistikat. Millist täpsemalt, selgub ilmselt töö käigus, aga mõned ideed:
	* kui palju lõppeesmärgist ja vahe-eesmärkidest (nt 1, 10, 50 jne tundi) on täidetud
	* töötundide arvu võrdlus erinevatel ajaperioodidel (nädala vaates näitab töötunde päevade kaupa, aasta vaates töötunnid kuude kaupa)
	* graafik, mis näitab, kuidas töötatud tundide arv on aja jooksul suurenenud
